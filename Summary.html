Web Programming Fifteen Puzzle

    Members

  • Doudou Sy Gueye: Leader, Programmer, Designer, Tester

  • Benjamin Pham: Programmer, Designer, Tester

Web Programming Fifteen Puzzle is a short game the user can play to pass time. it allows the user to choose what board they would like to play with and they have a clock that will run as they start the game and will terminate if they don't complete the game. If the user completes the game and wins they will be prompted that they won.
